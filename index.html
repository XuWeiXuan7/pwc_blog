🔑 Gitee API 被限流？点击设置 Access Token

Gitee 匿名 API 每小时只能请求约 60 次，很容易被限流返回 403。设置个人 Token 后可提升到 5000 次/小时。

  1. 打开 Gitee 个人令牌设置页
  2. 点击「生成新令牌」→ 名称随便写，勾选 projects、issues、user_info 权限即可
  3. 复制生成的 Token（以 a9 或 c1 等开头的一串字符），粘贴到下方
